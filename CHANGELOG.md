# Changelog

## 2.0.0

- Reworked the question bank into lazy-loaded course files under `assets/data/`.
- Added grade mode for elementary grades 1-6 and Kanken mode for levels 10-3.
- Added the shared `DrillQuestion` schema with 10 kanji categories.
- Added weighted question picking: 40% new, 40% weak, 20% review.
- Added review interval targeting for 1, 3, 7, and 14 days.
- Expanded learning history with accuracy, answer time, consecutive mistakes, last answered date, and consecutive correct count.
- Added study modes, answer explanations, improved answer button states, and enhanced results views.
