# Future Features

The Ver2.0 structure is intended to support these additions without large rewrites:

- AI explanations: add an explanation provider behind the answer screen.
- Voice reading: attach audio metadata or use a TTS service per `DrillQuestion`.
- Handwriting recognition: replace the writing completion button with a recognition service.
- Parent dashboard: aggregate `ProgressState` into child profiles.
- Rankings and online battle: sync profile summaries to a backend.
- Event questions: add new course asset files and route them through `StudyCourse`.
- Daily missions and calendar: extend `DailyChallenge` and store date-indexed activity.
- PDF/CSV output: export `ResultSummary` and `WeakItem`.
- Firebase/cloud sync: implement a storage adapter beside Hive.
